void drawQuad()
{

    glVertex3f(-50.0f,  0.0f, 0.0f);

}

03: Buffering

  • So far your application have been single buffered.

  • This is configured in main with GLUT_SINGLE

  glutInitDisplayMode(GLUT_SINGLE | GLUT_RGB | GLUT_DEPTH);
  • and in renedScene() where the single buffer is flushed:
void renderScene(void)
{
  glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);

  starWithoutEdges();

  glFlush();
}
  • For complex models and animations it may be preferable to switch to double buffering. This involves specifying the GLUT_DOUBLE display mode:
  glutInitDisplayMode(GLUT_DOUBLE | GLUT_RGB | GLUT_DEPTH);

  • and using this methdod:

  • glutSwapBuffers

  • instead of

  • glFlush

  • This would be the default for many applications.

  • In some circumstances you may require single buffering.
